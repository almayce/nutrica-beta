# Питание
- разнообразное

### Духовное
- вера
- воздержание
- аскетизм
### Душевное
- медитация
- дневник
### Телесное
- 4х часовое окно приема пищи
- интервальное питание
- интервальное голодание 48 часов каждые 2 недели
- тишина во время еды
